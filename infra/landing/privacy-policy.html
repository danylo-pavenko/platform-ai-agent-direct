Коротко: Direct AI Agents — це платформа, яка дозволяє підприємцям підключати AI-агента до своїх сторінок в Instagram для автоматизованого спілкування з клієнтами у Direct Messages. Ми обробляємо повідомлення від імені наших клієнтів-підприємців, відповідно до дозволів Meta Platform. Ми не продаємо ваші дані.

01

Про нас та сферу дії

Direct AI Agents — продукт компанії DEP Software (далі — «ми», «нас», «наша компанія»). Платформа надає послуги AI-агентів для автоматизованого спілкування з клієнтами через Instagram Direct Messages (DM), на основі промптів та джерел даних самих клієнтів-підприємців.

Ця Політика конфіденційності поширюється на:

  • клієнтів-підприємців (магазини, бренди, агенції), що використовують платформу Direct AI Agents;
  • кінцевих користувачів Instagram, які спілкуються з AI-агентом у Direct Messages;
  • відвідувачів вебсайту direct-ai-agents.com.

Платформа інтегрована з Meta Platform (Facebook / Instagram), що регулюється також Політикою конфіденційності Meta та Політикою платформи Meta для розробників.

02

Яку інформацію ми збираємо

Від клієнтів-підприємців:

  • Контактна інформація (ім'я, email, назва компанії).
  • Дані для інтеграції з Instagram (токени доступу до сторінок Meta/Instagram).
  • Каталог товарів, системні промпти, база знань, які використовуються для навчання AI-агента.
  • Платіжна інформація (обробляється зовнішнім платіжним провайдером).

Від кінцевих користувачів Instagram (через Meta Platform Webhooks):

  • Instagram User ID (IGSID) — унікальний ідентифікатор, наданий Meta.
  • Вміст повідомлень (Direct Messages), надісланих на сторінку підприємця.
  • Ім'я профілю та фото профілю Instagram (якщо доступне через Meta API).
  • Метадані повідомлень: час, статус прочитання.

Технічні дані:

  • Журнали запитів (IP-адреса, user-agent, час) — для безпеки та відлагодження.
  • Інформація про Webhook-підписки Meta.

Важливо: Ми збираємо дані кінцевих користувачів Instagram виключно від імені клієнта-підприємця в рамках авторизованих дозволів Meta Platform. Ми виступаємо як обробник даних (Data Processor), а клієнт-підприємець — як Контролер даних (Data Controller).

03

Дозволи Instagram / Meta Platform

Для роботи платформи клієнти-підприємці надають нашому застосунку такі дозволи Meta Platform:

  • instagram_manage_messages

    Читання вхідних DM та надсилання відповідей від імені Instagram-сторінки клієнта. Це основний дозвіл для роботи AI-агента.

  • pages_messaging

    Надсилання повідомлень через Messenger API для підключених Facebook-сторінок.

  • instagram_basic

    Читання базової інформації про Instagram-акаунт підприємця (назва, ID, фото профілю) для відображення в адміністративній панелі.

  • pages_read_engagement

    Читання базових показників взаємодії із сторінкою (необхідно для підтвердження підключення до Meta API).

  • pages_manage_metadata

    Підписка та управління Webhook-подіями сторінки для отримання повідомлень в реальному часі.

Webhook-події, на які ми підписуємось:

  • messages — вхідні та вихідні DM-повідомлення
  • messaging_seen — статус прочитання повідомлень
  • messaging_postbacks — взаємодія з кнопками (postback)

Дані, отримані через ці дозволи, використовуються виключно для надання сервісу — автоматизованих AI-відповідей у Direct Messages. Вони не використовуються для реклами, не передаються третім сторонам без згоди підприємця та не зберігаються довше, ніж це необхідно для надання послуги.

04

Як ми використовуємо дані

  • Генерація AI-відповідей. Вміст повідомлень передається до AI-моделі (Claude від Anthropic) разом із промптом та базою знань клієнта-підприємця для генерації релевантної відповіді.
  • Обробка замовлень. Структуровані дані замовлень (товар, розмір, контакт, адреса) передаються до CRM-системи клієнта (KeyCRM) для подальшого опрацювання менеджером.
  • Аналітика для підприємця. Агреговані метрики (кількість діалогів, конверсія, час відповіді) відображаються в адміністративній панелі клієнта.
  • Безпека та підтримка. Журнали використовуються для виявлення зловживань та відлагодження технічних проблем.

Ми не використовуємо дані для таргетованої реклами, не передаємо їх для маркетингових цілей та не аналізуємо контент для будь-яких цілей, не пов'язаних із наданням сервісу.

05

Передача даних третім сторонам

Ми співпрацюємо з такими сторонніми постачальниками послуг:

  • Anthropic (Claude AI) — постачальник AI-моделі для генерації відповідей. Вміст повідомлень передається до Anthropic API для обробки. Anthropic має власну Політику конфіденційності.
  • KeyCRM — CRM-система, яку використовують наші клієнти-підприємці. KeyCRM має власну інтеграцію з Meta Platform і є незалежним обробником даних. Дані замовлень передаються до KeyCRM відповідно до налаштувань клієнта. Дивіться Політику конфіденційності KeyCRM.
  • Хмарний провайдер (EU) — сервери для зберігання даних у зоні eu-central-1.

Ми не продаємо, не обмінюємо та не передаємо персональні дані комерційним третім сторонам. Передача відбувається лише в обсязі, необхідному для надання сервісу, та на підставі укладених угод про обробку даних (DPA).

06

Зберігання та безпека

  • Розташування: Усі дані зберігаються на серверах у Євросоюзі (eu-central-1). Кожен клієнт-підприємець отримує ізольований інстанс із власною базою даних.
  • Шифрування: Передача даних захищена TLS 1.3. Паролі та токени зберігаються у хешованому вигляді.
  • Термін зберігання: Дані повідомлень зберігаються максимум 90 днів або відповідно до налаштувань клієнта-підприємця. Токени доступу Meta анульовуються при відключенні інтеграції.
  • Видалення: Після розірвання договору з клієнтом-підприємцем усі дані видаляються протягом 30 днів.
  • Доступ: До даних мають доступ лише уповноважені співробітники в рамках виконання своїх обов'язків.

Ваші права

Відповідно до GDPR та застосовного законодавства ви маєте такі права:

  • Право на доступ — отримати інформацію про те, які ваші дані ми обробляємо.
  • Право на виправлення — виправити неточні або неповні дані.
  • Право на видалення («право бути забутим») — видалити ваші дані в строк 30 днів.
  • Право на заперечення — заперечити проти обробки ваших даних.
  • Право на переносимість — отримати ваші дані у машиночитаному форматі (CSV/JSON).

Для реалізації прав надішліть запит на help@depsoftware.com. Ми відповімо протягом 30 днів.

Якщо ви є кінцевим користувачем Instagram, ви також можете керувати своїми даними через налаштування Facebook/Instagram або звернутися до Meta безпосередньо.

Файли Cookie

Наш вебсайт використовує лише технічно необхідні cookie (сесія, налаштування мови). Ми не використовуємо cookie для реклами або відстеження третіми сторонами.

Адміністративна панель (admin) використовує localStorage для зберігання токену сесії. Ці дані не передаються третім сторонам.
