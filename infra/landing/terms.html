01

Сторони та предмет договору

Ці Умови користування (далі — «Умови») укладаються між DEP Software (розробник та власник платформи Direct AI Agents; CEO Данило Павенко; ceo@depsoftware.com) та юридичною або фізичною особою, що реєструється на платформі або використовує сервіс (далі — «Клієнт»).

Платформа Direct AI Agents (direct-ai-agents.com) — це SaaS-рішення, що надає AI-агентів для автоматизованого спілкування з клієнтами в Instagram Direct Messages. AI-агенти функціонують на основі системних промптів та джерел даних (каталог, FAQ, правила), які Клієнт надає самостійно.

Реєструючись або використовуючи сервіс, ви підтверджуєте, що прочитали, зрозуміли та приймаєте ці Умови та Політику конфіденційності.

02

Як працює сервіс

Direct AI Agents надає Клієнту ізольований AI-агент, що:

  • Підключається до Instagram-сторінки Клієнта через офіційний Meta Platform API (Facebook Login for Business).
  • Отримує повідомлення від підписників через Webhooks Meta у реальному часі.
  • Генерує відповіді на основі системного промпту та бази знань, наданих Клієнтом (каталог товарів, FAQ, tone of voice, правила ескалації).
  • Надсилає відповіді від імені Instagram-сторінки Клієнта.
  • Передає структуровані дані замовлень до CRM-системи Клієнта (за наявності інтеграції з KeyCRM або іншою системою).
  • Ескалює складні або чутливі запити до менеджера Клієнта через Telegram-бот або іншими каналами.

AI-застереження: Відповіді AI-агента генеруються автоматично на основі наданих Клієнтом даних. DEP Software не несе відповідальності за зміст відповідей, якщо вони суперечать наданим Клієнтом промптам або даним. Клієнт несе повну відповідальність за налаштування агента та його відповідність чинному законодавству.

05

Заборонене використання

Забороняється використовувати Direct AI Agents для:

  • Розсилання спаму, небажаних рекламних повідомлень або масових повідомлень без згоди одержувачів.
  • Продажу або просування товарів/послуг, що порушують законодавство або правила Meta (зброя, наркотики, дискримінаційні товари тощо).
  • Введення кінцевих користувачів в оману щодо наявності людини-оператора без відповідного розкриття.
  • Збору персональних даних кінцевих користувачів для цілей, не пов'язаних із наданою послугою.
  • Будь-якої діяльності, що порушує чинне законодавство, права інтелектуальної власності або конфіденційність.
  • Перепродажу або передачі доступу до платформи третім особам без письмової згоди DEP Software (крім White-Label агентств).

Порушення може призвести до негайного призупинення або видалення акаунту без повернення коштів.

06

Інтеграція з Meta / Instagram

Direct AI Agents використовує офіційний Meta Platform API для підключення до Instagram та Facebook. Для функціонування сервісу Клієнт надає нашому застосунку такі дозволи Meta:

  • instagram_manage_messages — для читання та надсилання DM.
  • pages_messaging — для Messenger API.
  • instagram_basic, pages_read_engagement, pages_manage_metadata — для підключення та управління Webhook.

Клієнт може відкликати ці дозволи в будь-який момент через налаштування Facebook/Instagram. Після відкликання дозволів агент припинить роботу. DEP Software не зберігає токени доступу довше, ніж це необхідно для надання сервісу.

Доступність та функції Meta API можуть змінюватись відповідно до рішень Meta Platforms Inc. DEP Software не несе відповідальності за зміни в API, що впливають на роботу сервісу.

Платформа підтримує інтеграцію з KeyCRM — CRM-системою з власною інтеграцією з Meta. При підключенні KeyCRM дані замовлень (товар, кількість, контакт, адреса) передаються безпосередньо до KeyCRM. Використання KeyCRM регулюється окремими умовами цього сервісу.

Дані та конфіденційність

Обробка персональних даних регулюється нашою Політикою конфіденційності, яка є невід'ємною частиною цих Умов.

Клієнт виступає Контролером даних (Data Controller) стосовно персональних даних своїх клієнтів (кінцевих користувачів Instagram). DEP Software виступає Обробником даних (Data Processor) і обробляє такі дані виключно відповідно до інструкцій Клієнта.

За запитом DEP Software підписує Угоду про обробку даних (DPA). Запити надсилати на help@depsoftware.com.

08

Оплата та тарифи

  • Пробний період: 14 днів безкоштовно, без введення платіжних даних.
  • Тарифи: деталі доступні на сторінці Тарифи. Ціни в EUR, виставляються щомісяця.
  • Оплата: рахунки виставляються на початку кожного розрахункового місяця. Несплата протягом 7 днів призводить до призупинення акаунту.
  • Повернення коштів: не здійснюється за вже оплачені місяці. Виключення можливі за рішенням DEP Software у разі задокументованих технічних проблем з боку платформи.

Скасування та видалення даних

  • Усі дані Клієнта (промпти, повідомлення, токени) видаляються протягом 30 днів після скасування.
  • Клієнт може запросити експорт своїх даних у форматі CSV/JSON до видалення.
